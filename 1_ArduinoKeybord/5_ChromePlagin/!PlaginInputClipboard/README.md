# Плагин для вставки из буфера обмена

## Установка
1. Скачайте все файлы плагина в одну папку
2. Откройте Chrome и перейдите на `chrome://extensions/`
3. Включите "Режим разработчика" (правый верхний угол)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с плагином

## Использование
- **Двойной клик** по любому текстовому полю → автоматическая вставка текста из буфера + эмуляция нажатия Enter
- **Ctrl+Shift+V** → вставка в активное текстовое поле + эмуляция нажатия Enter

## Работает с:
- Обычные текстовые поля (`<input type="text">`)
- Поля поиска (`<input type="search">`)
- Email и URL поля
- Парольные поля
- Многострочные текстовые области (`<textarea>`)
- Редактируемые элементы (`contenteditable`)

## Особенности
- Зеленая анимация при успешной вставке
- Красная анимация при ошибке
- Поддерживает замену выделенного текста
- Сохраняет позицию курсора
- Автоматическая эмуляция нажатия Enter после вставки
- Поддержка отправки формы при нажатии Enter

## Новый функционал:
После вставки текста автоматически эмулируется нажатие клавиши Enter, что:
1. Вызывает стандартные обработчики событий keydown/keyup
2. Автоматически отправляет форму если элемент находится внутри нее
3. Активирует поиск в поисковых полях
4. Выполняет навигацию в адресной строке
